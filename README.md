## 🎮 功能特点

### **一键参与功能**
- 🎲 **大号按钮**：140px × 50px 大按钮放置在每行右侧外部
- ✅ **智能状态**：自动识别已参与/未参与状态
- ↩️ **取消参与**：已参与按钮可点击取消参与
- 💰 **积分同步**：实时更新导航栏 P 点数显示

### **图标控制功能**
- 🔍 **大小调节**：支持 2.5x/2x/1.5x 三种倍率切换
- 🎛️ **控制面板**：左下角悬浮面板，显示处理图标数量
- ⌨️ **键盘快捷键**：
  - `Ctrl+1`：小图标 (1.5x)
  - `Ctrl+2`：中图标 (2.0x) 
  - `Ctrl+3`：大图标 (2.5x)

### **界面优化**
- 📌 **固定导航栏**：顶部导航栏始终漂浮显示
- ⚪ **P点数美化**：白色文字，无边框设计
- 🎨 **美观效果**：渐变背景、毛玻璃效果、增强阴影
- ⚡ **即时反馈**：按钮悬停动画、点击波纹效果

## 📦 安装方法

### **方法一：一键安装**
1. 安装油猴脚本管理器：
   - [Tampermonkey](https://www.tampermonkey.net/) (推荐)
   - [Violentmonkey](https://violentmonkey.github.io/)

2. 点击安装链接：**(https://github.com/xie7/SteamGifts_Enhanced2026/raw/refs/heads/main/SteamGifts_Enhanced.user.js)**

### **方法二：手动安装**
1. 复制完整脚本代码
2. 在脚本管理器中创建新脚本
3. 粘贴代码并保存
4. 访问 SteamGifts 网站启用

## 🛠️ 使用方法

### **参与抽奖**
1. 访问 SteamGifts 网站
2. 每行右侧会显示大号参与按钮
3. 点击"🎲 参与 (P)"按钮参与抽奖
4. 已参与的抽奖显示为"✅ 已参与"按钮，可点击取消

### **图标大小控制**
1. 页面左下角会显示图标控制面板
2. 选择图标大小：
   - **大**：2.5倍放大
   - **中**：2.0倍放大 (默认)
   - **小**：1.5倍放大
3. 使用键盘快捷键快速切换

### **界面操作**
- 🌙 **日夜模式**：点击右上角图标切换主题
- 🖱️ **拖拽调整**：可调节控制面板位置
- 📱 **响应式**：适配各种屏幕尺寸

## ⚙️ 技术特性

- **智能检测**：自动识别 SteamGifts 页面结构
- **动态加载**：支持页面翻页时自动重新初始化
